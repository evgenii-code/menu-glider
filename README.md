# Menu with Glider

Mobile menu with smooth glider

## Stack

- Vue.js
- Pug
- SCSS
- CSS Variables
- Responsive Design
- Flexbox/Grid
- i18n

## Preview

https://evgenii-code.github.io/menu-glider/

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run serve
```
